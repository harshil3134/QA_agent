# interview_task_1

Minimal project demonstrating local document embeddings with Chroma and HuggingFace embeddings.

Quick start:

1. Install dependencies (in a virtualenv):

```bash
python -m pip install -r requirements.txt
```

2. Run the main script:

```bash
python main.py
```

Set `GROQ_MODEL` environment variable to pick a Groq model if needed.
